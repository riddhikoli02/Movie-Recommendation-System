
# Movie Data Analysis Project

This project analyzes movie ratings using Python and Google Colab.

## Dataset

The dataset used is the MovieLens dataset.

Download it from here: https://www.kaggle.com/datasets/parasharmanas/movie-recommendation-system

After downloading, place these files in the same folder as the notebook:

* movies.csv
* ratings.csv

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Analysis

* Most rated movies
* Average ratings per movie
* Ratings distribution
* Most popular genres
